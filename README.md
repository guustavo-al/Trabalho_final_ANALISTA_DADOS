Trabalho Final - Analista de Dados II - ADA


Este trabalho visa analisar dados extraidos do site Global Nuclear Power Tracker. 

https://globalenergymonitor.org/projects/global-nuclear-power-tracker/download-data/

Os dados representam o atual cenario global das usinas nucleares no mundo. Apresentados dados como Nome, Status, Data de construção, Produção de energia, Tipo de reator, e outros. 

O arquivo, .ipynb carrega o arquivo bruto. As primeiras linhas de código removem colunas que não são interessantes para a analise, por exemplo, latitude, longitude, cidade, URL do reator e outros. 

O restante do trabalho utiliza código em python com pandas para extrair dados da tabela. Os dados extraidos são comentados e analisados, sempre tomando em consideração o que os dados representam, não apenas o dado bruto.
